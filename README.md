# Riftlords: Nemesis Siege

A browser-playable 2D side-view base battle game inspired by Age of War with a lightweight nemesis commander layer.

## Features
- Real-time base-vs-base battles with unit spawning, gold economy, health bars, and floating damage numbers.
- 6 unit types: Militia, Spearman, Archer, Knight, Priest, Siege Ram.
- 8-territory campaign map with enemy commanders and modifiers.
- Nemesis system with persistent commanders stored in localStorage.

## Run locally
```bash
npm install
npm run dev
```
Then open the local URL printed by Vite.
